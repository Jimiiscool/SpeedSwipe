# SPEEDSWIPE!
A speedrun-based game that gives a set combination for the player to swipe using our main item: a card swiper!
The player will start with a timer that will start from the moment the user swipes. The player must move as fast as they can to beat the level. If they swipe it in the wrong direction, the player will have to start all over again with a new combination while the timer continues. The fastest swiper wins!
